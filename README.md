## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

### Installation


1. Get a free API Key at [https://www.deepl.com/en/docs-api/](https://www.deepl.com/en/docs-api/)
2. Clone the repo
   ```sh
   git clone https://github.com/babel-on/penpals.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create .env file at root of directory

5. Enter your API in `.env`
   ```js
   DEEPL_KEY='ENTER YOUR API';
   ```
6. Connect MongoDB Atlas database in `.env`
   ```js
   MONGO_URI='YOUR MONGO URI STRING';
   ```
7. Set up JWT secret in `.env`
   ```js
   JWT_KEY='your string of choice';
   ```

<!-- USAGE EXAMPLES -->
## Usage

Sign up for an account, and talk to users across the world! Penpals automatically translates incoming messages to the language of your choice.
Multilingual and want to change languages? No problem. Penpals will also store your previous messages and re-translate them to your new language.
